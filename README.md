Code associated with Late Holocene Stabilization of Conway Ice Ridge
Andrew O. Hoffman, Paul T. Summers, Jenny Suckale, Knut Christianson, Ginny Catania, and Howard Conway.

Installation of CVX (https://cvxr.com/cvx/doc/install.html) and Matlab is needed to run this code.
All assocaited data for making the figures in the paper will be published a linked repository. 

Paul Summers developed ran the model, and can be reached with any questions.
